Projeto lista de tarefas
1 - criar projeto em next.js
npx create-next-app@latest .
 
2 - criar repositório no git hub para o projeto
git init
git add
git commit
git branch -m main
git remote add origin 'colocar o link do projeto no github'
git push origin main
 
3 - instalar biblioteca de componentes shadcn https://ui.shadcn.com
npx shadcn@latest init

3.1 - instalar componente do shadcn no projeto
Vá até installation, copie o código e cole no terminal. Exemplo o componente button: npx shadcn@latest add button

3.2 - o shadcn utiliza os ícones da Lucide https://lucide.dev
